Сервис для предоставления/реализации облачных вычислений
Задача для распределения: оптимизационный поиск по сетке гиперпараметров 
GridSearch моделей библиотеки scikit-learn

Со стороны потребителя: через специальное API в коде питона запрашиваются 
устройства для выполнения поиска по сетке
Входные данные: в качестве параметра передаётся модель, тренировочные данные
Выходные данные: набор обученных моделей с комбинациями гиперпараметров

Со стороны исполнителя: при получении запроса на выполнение разворачивается контейнер с питоном 
для тренировки модели
Входные данные: конкретная модель с конкретными данными обучения
Выходные данные: обученная модель, её метрики

Промежуточный узел: сервер принимает запросы на вычисление, распределяет модели по исполнителям,
собирает результаты вычисления и отправляет их обратно потребителю
